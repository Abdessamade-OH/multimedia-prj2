Project Overview

But : Développer une application Web pour un système d'indexation et de recherche d'images.

  • Charger, télécharger, supprimer des images
  • Organiser les images en catégories
  • Créer des nouvelles images avec transformations
  • Consommer des services API pour descripteurs d'images
  • Retourner des images similaires (avec et sans pertinence)
Important : Méthode de retour de pertinence : Query-point movement method

Features of the Application

  • Image Upload/Download/Delete
  • Image Categorization
  • Image Transformation
  • Content Descriptors
  • Search with Relevance Feedback
